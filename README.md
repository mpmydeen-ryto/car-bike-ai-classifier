# 🚗🏍️ Car vs Bike AI Classifier

An AI-powered web application that classifies uploaded vehicle images as either **Car** or **Bike** using a TensorFlow deep learning model.

## 🌟 Features

- Upload any vehicle image
- AI predicts whether it is a Car or Bike
- Displays prediction confidence score
- Built with TensorFlow and Python
- Deployed online using Hugging Face Spaces

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/mpmydeen-ryto/car-bike-ai-classifier.git
cd car-bike-ai-classifier
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
